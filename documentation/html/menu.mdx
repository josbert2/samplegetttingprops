# Tailwind CSS Dropdown Menu

Use our responsive Tailwind CSS dropdown menu, that can take the user anywhere on your web app!

Our dropdown menu displays a list of choices on temporary surfaces. It appears when users interact with a button, action, or other control. When a user clicks/hovers over the parent element, the dropdown menu appears, presenting a vertically stacked list of actions.

See below our dropdown menu examples that you can use in your Tailwind CSS project. The example comes in different colors and styles, so you can adapt them easily to your needs.

To create a menu you can use the <Link href="/docs/html/popover#popover-trigger-data-attribute"><Code>data-popover-target="`{menuName}`"</Code></Link> and <Link href="/docs/html/popover#popover-data-attribute"><Code>data-popover="`{menuName}`"</Code></Link> data attributes.

<br />

## Nested Dropdown Menu

Use this component example if you want to implement a nested dropdown menu structure. You can do this by adding a  <Code>data-popover-target="nested-menu"</Code> attribute on one of the menu items, so that selecting this item will open another menu adjacent to the current one.

You can create nested/multi-level menus by setting the <Link href="/docs/html/popover#popover-trigger-data-attributes"><Code>data-popover-nested="true"</Code></Link> data attribute to the menu trigger element.
